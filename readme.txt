A companion plugin for WordPress themes by MysticalThemes.

== Description ==

Mystical Companion is a companion plugin for the themes from MysticalThemes. The plugin contains some elementor elements and meta options for configuring theme.

== Installation ==

= Minimum Requirements =

* WordPress 4.7 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Make sure you have installed and activated the 'Elementor' plugin before installing the plugin then,
2. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to Pages > Add New
5. Press the 'Edit with Elementor' button.
6. Now you can drag and drop widgets under `Mystical Elements` from the left panel onto the content area, as well as add new sections and columns that make up the page structure.
